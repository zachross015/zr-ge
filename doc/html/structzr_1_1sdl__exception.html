Detailed Description

Class of exceptions which deals with errors that occur within the SDL codebase.

It primarily does so by reiterating whatever error message is passed to the class, along with the message passed along by SDL_GetError. Part of the big reason for creating such a wrapper was to include stronger typing and exception handling as is provided by more modern c++, which is what this struct helps to facilitate.
